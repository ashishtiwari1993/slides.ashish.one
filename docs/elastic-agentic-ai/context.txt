## Presentation Mechanics

### Slide System
- Each slide is a `<div class="slide">` inside a `.deck` container
- Slides are absolutely positioned, full viewport, with `opacity: 0` and `transform: translateX(60px)` by default
- Active slide gets `.active` class → `opacity: 1; transform: translateX(0)`
- Exiting slides get `.exit-left` or `.exit-right` classes for directional transitions
- Slides have `overflow-y: auto` for internal scrolling when content exceeds viewport

### Navigation
- **Keyboard**: Arrow keys, Space, PageDown/PageUp, Home/End
- **Click**: Left half = previous, right half = next (exclude interactive elements)
- **Touch/Swipe**: Horizontal swipe detection (dx > dy && dx > 40px threshold)
- **Fullscreen**: F key or button in bottom-right corner

### Fixed UI Elements
- **Header**: Fixed top bar (40px) with Elastic logo, semi-transparent dark background with backdrop-filter blur
- **Progress bar**: 2px gradient bar (teal→blue) below header, animates with shimmer effect
- **Slide counter**: Bottom center, JetBrains Mono, shows "X / Y"
- **Presenter notes**: Hidden panel at bottom, toggled with N key, reads `data-notes` attribute from each slide

### Animation System
- Auto-tagging JS system (ANIM_MAP) — scans each slide for elements matching CSS selectors and assigns animation classes
- Animation types: `fadeInUp`, `fadeInDown`, `fadeInLeft`, `fadeInRight`, `scaleIn`, `blurIn`
- Duration variants: `anim-fast` (0.4s), `anim-normal` (0.6s), `anim-slow` (0.85s)
- Stagger delays: `anim-d0` through `anim-d10` (80ms increments)
- Animations reset on each slide entry via JS `resetAnimations()` function
- Card hover: lift + glow (translateY(-4px) + box-shadow)
- Highlight text: shimmer gradient animation on `.highlight` class (teal gradient, background-clip text)
- Title slide: Canvas-based particle system with Elastic brand colors and connecting lines

### Responsive Design (3 breakpoints)
- **1024px**: Stack two-col to single, pipeline goes vertical
- **768px**: Full mobile layout — single columns, tighter padding (12–16px), smaller fonts, swipe hint on counter. Use CSS attribute selectors `[style*="..."]` with `!important` to override inline styles that don't stack
- **480px**: Extra tight — smallest font sizes, compressed spacing
